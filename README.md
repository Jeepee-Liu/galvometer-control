# galvometer-control

LabVIEW program for controlling a 2-D galvometer and reading data from microscope.
Avaliable version: after LabVIEW 13.0 (2013), including LabVIEW 2013, 2014, 2015.

To run the program, open 'syncDAQ_main.vi'.
